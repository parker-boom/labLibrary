# Brand System

## Brand Idea

Lab Library is a retro arcade portal for campus AI event ideas. It should feel like a small machine that helps hosts pick a format, steal a workflow, and run something real.

## Core Feeling

- Playful, but not childish.
- Futuristic, but not sterile.
- Student-native, not corporate.
- Physical and responsive, not flat.
- Simple enough to understand in 10 seconds.

## Visual Tokens

```css
:root {
  --lab-black: #050505;
  --lab-yellow: #d8b914;
  --lab-white: #f7f3e8;
  --lab-gray: #8e8a7a;
  --lab-ink: #11100b;
}
```

## Type Direction

- Display headings: chunky geometric / arcade-adjacent, similar to Space Grotesk in weight and confidence.
- Small UI text: clean mono or mono-adjacent, similar to IBM Plex Mono.
- Do not use fake-pixel body text for long reading surfaces.

## Layout Rules

- Three-door homepage: Event Tracks, Student Use Cases, Past Lab Events.
- Cards should feel like physical game panels.
- Use one colorful pixel-style icon per section/card.
- Keep the palette restrained. The page should not become rainbow.
- Real event photos should feel like proof, not decoration.

## Avoid

- Generic SaaS dashboard styling.
- Corporate resource-hub language.
- Category-chip clutter.
- Random arrows.
- Bulky 3D icons.
- Dense explanatory text on the main browsing surfaces.
