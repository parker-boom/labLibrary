# Generated Assets

Generated pixel-art assets used by the deployed app live here. Source/candidate sheets that are not referenced by the app are intentionally not shipped in `public/` so the deployable asset set stays small.

## Home Icons

- `home-icons/event-tracks.png`
- `home-icons/student-use-cases.png`
- `home-icons/past-lab-events.png`

Prompt summary: three simple arcade-inventory home icons on a flat chroma-key background: an arcade portal, a joystick controller, and an instant camera. Backgrounds were removed and each icon was normalized onto a transparent square canvas.

## Home Hover Loops

- `home-hover-loops/portal-loop-sheet.png`
- `home-hover-loops/joystick-loop-sheet.png`
- `home-hover-loops/camera-loop-sheet.png`

Prompt summary: three six-frame horizontal pixel-art hover animation sheets matching the approved home portal, joystick, and instant-camera icons. Chroma key was removed locally, then each loop was normalized into six transparent `512px` frames in a single horizontal sprite sheet. These run on hover only.
